# 英语听写工具 Pro

## 🎯 功能简介

这是一个简洁高效的英语单词听写训练工具，支持智能发音、错题管理、进度统计等功能。

## 🚀 快速开始

### 1. 添加单词
- 在"添加新单词"框中输入英文单词，每行输入一个单词，点击"导入"
- 系统会自动获取单词的释义、音标和词性

### 2. 选择听写范围
- **全部单词**：听写词库中的所有单词
- **选中单词**：只听写勾选的单词（在单词列表中勾选）
- **错题本**：只重复听写做错的单词

### 3. 设置听写参数
- **播放顺序**：随机或顺序播放
- **播放次数**：每个单词播放1-3次
- **播放间隔**：单词间的等待时间
- **朗读速度**：慢速到快速可调
- **显示选项**：可选择显示/隐藏单词、释义、音标
- **输入模式**：应用内输入或线下听写

### 4. 开始听写
- 点击"开始听写"按钮
- 在输入框中输入听到的单词
- 按回车键或点击"提交答案"
- 系统自动判断对错并记录

### 5. 错题复习
- 做错的单词自动进入错题本
- 可以随时查看错题本，针对性练习
- 答对的单词会自动减少错误次数

## 📊 功能特点

✅ **智能发音** - 使用浏览器语音合成技术，发音清晰标准  
✅ **自动翻译** - 自动获取单词释义并翻译成中文  
✅ **错题管理** - 自动记录错题，支持针对性复习  
✅ **进度统计** - 实时显示听写进度和正确率  
✅ **灵活选择** - 支持选择部分单词听写  
✅ **数据持久化** - 所有数据保存在本地，关闭页面不丢失  
✅ **批量操作** - 支持批量导入、全选、清空等操作  
✅ **响应式设计** - 支持电脑和移动设备  

## ⌨️ 快捷键

- **Ctrl + Enter**：开始听写
- **Enter**：提交答案（听写中）
- **Esc**：结束听写

## 🔧 技术说明

- **数据存储**：使用浏览器本地存储（localStorage）
- **语音合成**：Web Speech API
- **词典API**：Free Dictionary API
- **翻译服务**：Google Translate API

## 📂 文件说明

- `index.html` - 主页面
- `app.js` - 核心功能脚本

## ⚠️ 注意事项

1. 首次添加单词时需要网络连接获取释义
2. 建议使用 Chrome、Edge 等现代浏览器
3. 数据保存在浏览器本地，清理浏览器数据会清空词库
4. 如无法发音，请检查浏览器语音合成权限
